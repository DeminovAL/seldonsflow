# Политика конфиденциальности для SandBox

**Дата вступления в силу:** 09.04.2026

## 1. Какие данные мы собираем
Наше приложение запрашивает доступ к вашему аккаунту Pinterest через официальный API. Мы получаем и обрабатываем только те данные, которые необходимы для анализа ваших досок. К ним могут относиться:
- Информация о ваших досках и пинах.
- Ваше имя пользователя и идентификатор аккаунта Pinterest.

Мы **НЕ** собираем и не храним ваш пароль от Pinterest, данные вашей банковской карты или другую личную информацию.

## 2. Как мы используем данные
Мы используем полученные данные исключительно для того, чтобы показывать вам аналитику по вашим доскам. Мы не продаем и не передаем ваши данные третьим лицам.

## 3. Хранение и безопасность
Мы не храним данные на своих серверах. Ваш ключ доступа (токен) используется только во время сессии для прямого взаимодействия с API Pinterest.

## 4. Ваши права
Вы можете в любой момент отозвать доступ приложения к вашему аккаунту Pinterest в настройках вашего профиля.

## 5. Контакты
Если у вас есть вопросы по этой политике, свяжитесь со мной: ссылка на профиль GitHub -deminoval (https://github.com/DeminovAL).
